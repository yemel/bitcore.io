new Unit(amount, code)

Utility for handling and converting bitcoins units. The supported units are
BTC, mBTC, bits (also named uBTC) and satoshis. A unit instance can be created with an
amount and a unit code, or alternatively using static methods like {fromBTC}.
You can consult for different representation of a unit instance using itâ€™s
{to} method, the fixed unit methods like {toSatoshis} or alternatively using
the unit accessors.

Params

  • amount Number - The amount to be represented
  • code String - The unit of the amount

Returns: Unit - A new instance of an Unit
